Class 2 - Logic & Argument

Date Wed 24 August 2016 Tags logic / argument

Today in class we’ll continue discussing the basic features of an argument, including their basic logical structure. We’ll look at some examples of the different forms an argument can take, and we will practice extracting an argument from a text.

Readings

  • What is an argument?
  • Describing an argument
  • Examples of good and bad argument forms

Videos

The following are some short videos covering many of the concepts discussed in the readings above, and which I’ll cover again in lecture. The concepts of truth, validity, soundness, and deductive argument are central to all of the material we cover in this course, so it is important that you have a decent grip on them.
